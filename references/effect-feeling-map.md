# Effect -> Feeling Mapping

Use this map to pick motion/visual systems that match intent.

## Dramatic / Cinematic
- Slow fade-ins (1-1.5s)
- Large scale transitions (0.9 -> 1)
- Dark backgrounds with spotlight gradients
- Parallax depth with restraint
- Full-bleed media moments

## Techy / Futuristic
- Subtle neon edge accents (not glow spam)
- Grid/pattern overlays
- Monospace accent typography
- Scramble/decode text moments
- Cyan/magenta/electric blue used sparingly

## Playful / Friendly
- Spring/bouncy easing
- Rounded surfaces
- Brighter/pastel palettes
- Float/bob loops on low-amplitude cycles
- Illustrated/soft geometric accents

## Professional / Corporate
- Fast, subtle transitions (200-300ms)
- Clean sans-serif hierarchy
- Navy/slate/charcoal foundations
- Precise spacing and alignment
- Minimal ornament, data clarity first

## Calm / Minimal
- Very subtle motion
- High whitespace
- Muted palettes
- Serif headline with sans body (when appropriate)
- Content-first framing

## Editorial / Magazine
- Strong typographic rhythm
- Pull quotes/callouts
- Image-text interplay
- Grid-breaking compositions with control
- Monochrome + one accent
